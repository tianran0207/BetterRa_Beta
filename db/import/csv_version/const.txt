//=============================================================
// 脚本 - 变数索引数据库（公共库）
//=============================================================
//
// 数据库结构:
// 标识	值 [类型]
//
// Alternatively:
// Identifier,Value[,Type]
//
// 01. Identifier   常量的惟一名称，不能与脚本命令、函数或变量同名。
// 02. Value   分配给常数的值。此值在服务器运行时无法更改。
// 03. Type    指定标识符是常量(0)还是参数(1)(可选，默认为0)。
//
// NOTE:  参数是特殊的，因为它们根据字段值中的指定ID检索特定的运行时值。
//        根据由脚本分配给参数的实现值将会影响运行时值，
//        比如Zeny(参见pc_readparam/pc_setparam)。
