//=============================================================
// 腳本 - 變數索引數据庫（公共庫）
//=============================================================
//
// 數据庫結构:
// 標識	值 [類型]
//
// Alternatively:
// Identifier,Value[,Type]
//
// 01. Identifier   常量的惟一名稱，不能与腳本命令、函數或變量同名。
// 02. Value   分配給常數的值。此值在服務器運行時無法更改。
// 03. Type    指定標識符是常量(0)還是參數(1)(可選，默認為0)。
//
// NOTE:  參數是特殊的，因為它們根据字段值中的指定ID檢索特定的運行時值。
//        根据由腳本分配給參數的實現值將會影響運行時值，
//        比如Zeny(參見pc_readparam/pc_setparam)。
